# Granfell - Ziko's Portfolio Website

Portfolio personal website untuk Ziko dengan informasi lengkap tentang biodata, kontak, media sosial, dan game profile.

## 📂 Struktur Folder

```
public/
├── home.html           # Halaman utama
├── biodata.html        # Informasi biodata Ziko
├── contact.html        # Halaman kontak
├── game.html           # Game ID & UID PUBG
├── infov1.html         # Informasi tambahan
├── medsos.html         # Link media sosial
├── ulasan1.html        # Formulir ulasan
└── favicon.ico         # Icon browser
```

## 📄 Halaman Tersedia

- **Home** - Halaman utama dengan menu navigasi
- **Biodata** - Informasi pribadi dan pengalaman
- **Contact** - Halaman kontak dengan video background
- **Game** - PUBG Mobile ID & UID
- **Info v1** - Informasi medsos & game
- **Media Sosial** - Link ke semua akun media sosial
- **Ulasan** - Formulir feedback

## 🎨 Fitur

- Responsive design untuk mobile & desktop
- Animasi halus dengan CSS
- Video background pada beberapa halaman
- Modern UI dengan gradient dan shadow effects
- Link navigasi antar halaman

## 🚀 Cara Menggunakan

1. Clone repository
2. Buka file `public/home.html` di browser
3. Navigasikan ke halaman lain melalui menu
